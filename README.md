# milktea-pos-inventory-barcode-system
A complete Milktea Point of Sale (POS) and Inventory Management System with Barcode Scanning built using HTML, JavaScript, and Firebase.  Features include product management, sales processing, inventory tracking, low stock monitoring, and real-time data synchronization for small businesses and school capstone projects.

## Features
- POS System
- Barcode Scan
- Inventory Management
- Firebase Ready

## Setup
1. Replace firebase config
2. Open index.html
3. Deploy with Firebase
